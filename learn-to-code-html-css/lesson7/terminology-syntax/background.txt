## Background Color
div {
  background-color: blue;
}

## Background Image
div {
  background-image: url("wallpaper.jpg");
}

## Background Repeat
Determines rather a background will repeat or not.

div {
  background-repeat: no-repeat;
}

## Background Position
Adjusts the position by a horizontal offset and a vertical offset.

div {
  background-position: 20px 10px;
}

## Shorthand
div {
  background: url("wallpaper.jpg") 20px 10px no-repeat;
}

## Background Gradient
div {
  background: liner-gradient(#fff, #000);
}

## Direction
div {
  background: linear-gradient(to right bottom, #fff. #000);
}

## Multiple Colors
div {
  background: linear-gradient(#fff, #000, #fff);
}

## Stops
div {
  background: linear-gradient(#fff, #000 80%, #fff);
}

##  Radial Pattern
div {
  background: radial-gradient(#fff #000);
}
